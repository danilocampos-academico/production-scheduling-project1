## Descrição do Projeto

Este projeto implementa e compara dois modelos matemáticos de sequenciamento em máquina única, ambos com objetivo de minimizar a soma ponderada dos tempos de conclusão.

Os modelos são:

- **F1 — Completion Time Variables**  
- **F4 — Assignment and Positional Date Variables**

A implementação é feita em **Julia + JuMP**, utilizando o solver **HiGHS**.

O projeto utiliza duas instâncias para testes, uma com 40 jobs e outra com 20 jobs.
